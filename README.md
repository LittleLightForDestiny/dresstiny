# quickstart

* copy env-config.example.ts to env-config.ts
* add your api key to env-config.ts

run:
```bash
npm install
npm run build-database
npm start
```

* open http://localhost:8080 in your browser


# export prod version
```bash
npm run build
copy ./database to ./dist/database
```
